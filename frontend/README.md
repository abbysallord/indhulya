## 💎 Overview
This is the frontend client for the Indhulya luxury e-commerce platform. Built with a focus on **"Quiet Luxury"**, it leverages modern web technologies to deliver an ultra-premium, interactive shopping experience.

## ✨ Key Features
- **Signature Aesthetics**: Uses Indhulya's authentic brand palette (Deep Maroon/Aubergine `#5C1218`, Gold `#E5B94E`, and Ivory `#FAF9F6`).
- **Aceternity UI Enhancements**: Implements premium micro-interactions including Text Generation effects, Infinite Scrolling Masks, and glowing interactive hover states.
- **AI Chatbot**: A custom, floating AI assistant built directly into the UI.
- **Responsive Architecture**: Fully responsive and mobile-first, ensuring an app-like feel across all devices.

## 🛠 Tech Stack
- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 3](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

First, install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. The application supports hot-reloading for immediate feedback.

## 📁 Core Directory Structure
- `app/`: Next.js App Router root layout, pages, and global CSS.
- `components/`: Modular UI elements (`Header.tsx`, `HeroCarousel.tsx`, `EditorialSection.tsx`, etc.).
- `public/`: Static brand assets and imagery.
