## 🕌 About The Project

Welcome to the **Indhulya Monorepo**. This repository houses the complete digital infrastructure for Indhulya—a high-end, Demifine jewellery brand specializing in authentic Temple, Antique, and One Gram Gold pieces.

This platform is engineered to deliver a world-class, **"Quiet Luxury"** aesthetic, pairing cinematic frontend experiences (powered by Aceternity UI and Framer Motion) with a robust, scalable backend architecture.

---

## 📂 Repository Architecture

This project is structured as a monorepo, cleanly separating the client-side experience from the server-side logic and database operations.

### 🌐 1. `frontend/` (The Client)
The `frontend` directory contains the Next.js application. It is responsible for the visual identity and user interaction.
- **Tech**: Next.js App Router, Tailwind CSS, Framer Motion.
- **Highlights**: Features a custom AI Chatbot, smooth scrolling navigation, and Aceternity UI text-reveal animations. 
- *[Read the Frontend Documentation](./frontend/README.md)*

### ⚙️ 2. `backend/` (The Server - *Coming Soon*)
The `backend` directory will house the API services, database models, and headless CMS integrations.
- **Scope**: Handles secure checkout, product inventory syncing, automated mailing, and webhook integrations.

## 🎨 Brand Identity Guidelines
To ensure consistency across future developments, adhere strictly to the Indhulya design tokens:

| Name | Hex Code | Purpose |
| :--- | :---: | :--- |
| **Deep Maroon** | `#5C1218` | Primary brand color, typography, buttons, footers. |
| **Royal Gold** | `#E5B94E` | Accents, borders, badges, hover glows. |
| **Pristine Ivory** | `#FAF9F6` | Primary background canvas. |

## 🚀 Quick Start
To launch the frontend locally:
```bash
cd frontend
npm install
npm run dev
```
